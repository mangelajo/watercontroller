HTTPS

      
Paste a cert + private key to enable HTTPS on port 443. Requires
        rebuilding the firmware with CONFIG_ESP_HTTPS_SERVER_ENABLE=y (off in default
        sdkconfig because of an unrelated qemu issue). Both fields empty = HTTP only.

API authentication

      

        
        
        When set, every POST/PUT request must
          carry Authorization: Bearer <token>. Empty = open access (only safe
          on a trusted LAN). Don't lose this — Factory reset is the only recovery.

Firmware update (OTA)

      
Pick a release-built watercontroller-firmware ELF and
        upload. The device writes it to the inactive OTA slot and reboots; if the new firmware
        fails its self-test the bootloader rolls back automatically.

      

        
        
        
      

      

        
      

    


    
    

      
Danger zone

      
Factory reset wipes the persisted config and reboots the device.
        Useful if WiFi credentials get the device into an unreachable state.
