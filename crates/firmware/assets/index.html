Sprinkler auto-off durations

When a sprinkler is turned on (manually, by schedule, or via MQTT), it auto-offs after this many minutes. Set to 0 to disable auto-off (manual stop only).

YAML default: 7 min.
YAML default: 5 min.

Saved values apply immediately to the running state machine — no reboot required. If a sprinkler is currently on, the new duration is measured against the existing turn-on timestamp.

Water valve timing

The motorized valve opens/closes the water circuit. After a close, the drain output (GPIO25) is held high to evacuate the pipes. The drain does not block subsequent commands — requesting "open" mid-drain stops the drain immediately and starts the open sequence. Set drain duration to 0 to skip the drain phase entirely.

Default 14 s — time the motor needs to fully open or close.
Default 300 s (5 min). 0 disables the drain phase.

HTTPS

Paste a cert + private key to enable HTTPS on port 443. Requires rebuilding the firmware with CONFIG_ESP_HTTPS_SERVER_ENABLE=y (off in default sdkconfig because of an unrelated qemu issue). Both fields empty = HTTP only.

Wireguard

⚠ Wireguard FFI is not yet wired in the firmware — these fields persist but the tunnel is not initialized.

API authentication

When set, every POST/PUT request must carry Authorization: Bearer <token>. Empty = open access (only safe on a trusted LAN). Don't lose this — Factory reset is the only recovery.

Firmware update (OTA)

Pick an app-only image (target/firmware/app.bin, produced by make app-image) and upload. The device writes it to the inactive OTA slot and reboots; if the new firmware fails its self-test the bootloader rolls back automatically.

Advanced (raw JSON)

Editable GET /api/config blob. Useful for fields the forms don't expose, or for backup/restore by copy-paste.

Danger zone

Factory reset wipes the persisted config and reboots the device. Useful if WiFi credentials get the device into an unreachable state.

Diagnostics (auto-refresh)

Live heap and per-task stack high-water marks from /api/diag. Refreshes every 3 seconds while this tab is open. Useful for spotting memory pressure (low min ever free) or tasks dangerously close to overflowing their stack (low stack free).
